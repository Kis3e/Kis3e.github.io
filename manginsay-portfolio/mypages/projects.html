BREWED COFFEE WEBPAGE

"Brewed Coffee" is a basic webpage project I created from scratch using HTML and CSS. The design was made using Figma, and though it hasn't been published or finalized yet, it showcases a clean and simple layout with a coffee image on the homepage. The site includes a navigation menu with links to the Home, About, and Services pages, along with a "Buy Now" button for potential users to explore products. The design aims to be warm and welcoming, perfect for a coffee shop, and easy for users to navigate.
